# Code of Conduct

willow-1.7 is a personal infrastructure project. Participation is expected to be respectful and constructive.

## Expected behavior

- Be specific and technical. This is an engineering project.
- Be direct. Brief, precise communication is valued over lengthy discourse.
- Assume good intent in ambiguous situations.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language of any kind.
- Spam or off-topic promotion.
- Attempts to manipulate the project's direction through social pressure rather than technical argument.

## Scope

This applies to GitHub Issues, Discussions, and Pull Requests.

## Enforcement

Violations will result in removal of the offending content and, if repeated, loss of access to the repository.

Contact: open a private GitHub issue or email the maintainer.
